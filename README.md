# 🍽️ Sistema de Gerenciamento de Restaurantes

Este projeto foi desenvolvido durante meus estudos de **Python na Alura**.  
O programa simula um sistema simples de gerenciamento de restaurantes no terminal.

## 📌 Funcionalidades

O sistema permite:

- Adicionar novos restaurantes
- Listar restaurantes cadastrados
- Alterar o estado do restaurante (Ativo / Inativo)

## 🧠 Conceitos utilizados

Durante o desenvolvimento foram aplicados diversos conceitos de Python, como:

- Funções para organizar o código
- Estruturas condicionais (`if`, `elif`, `else`)
- Tratamento de erros com `try` e `except`
- Uso da biblioteca `os` para limpar o terminal e melhorar a visualização

## ⚙️ Estrutura do programa

O código foi dividido em funções para facilitar a organização, como:

- Exibir nome do programa
- Mostrar menu principal
- Voltar ao menu
- Gerenciar restaurantes

## 🚀 Como executar o projeto

1. Clone o repositório
2. Abra o terminal na pasta do projeto
3. Execute o arquivo Python:

```bash
python app.py
